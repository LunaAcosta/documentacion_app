Calculadora de Interés Compuesto Ingeniería Económica

Introducción

La Calculadora de Interés Compuesto es una aplicación diseñada para calcular diferentes aspectos financieros relacionados con el interés compuesto. Con esta aplicación, puedes calcular el valor actual, el valor futuro, el tiempo y la tasa de interés compuesta.

Requisitos del Sistema

01

Windows, macOS, Linux

02

Python instalado (versión 3.x)

03

Bibliotecas: tkinter, customtkinter, PIL

Instalación

01

Descarga el archivo de código fuente

02

Instala las bibliotecas requeridas

Ejecuta el archivo principal de la aplicación (main.py) utilizando Python.

Uso de la Aplicación

Interfaz Principal

Al abrir la aplicación, verás la interfaz principal con un menú lateral y el área de trabajo principal donde se muestran los cálculos.

Menú Lateral

El menú lateral contiene diferentes opciones para realizar cálculos

  1. ✓ Valor Actual
  2. ✓ Valor Futuro
  3. ✓ Interés
  4. ✓ Información
Realizar Cálculos
  1. 1. Haz clic en una opción del menú lateral para seleccionar el tipo de cálculo que deseas realiza
  2. 2. Completa los campos de entrada con los valores necesarios para el cálculo
  3. 3. Haz clic en el botón "Calcular" para obtener el resultado
  4. 4. Si deseas borrar los valores ingresados, puedes hacer clic en el botón "Borrar"
Navegación

Puedes cambiar entre las diferentes opciones de
cálculo haciendo clic en los botones del menú lateral

Visualización de Resultados

Los resultados de los cálculos se mostrarán en el área de
trabajo principal de la aplicación.

Notas Adicionales

La aplicación maneja automáticamente los errores de entrada del usuario y muestra mensajes de error cuando sea . Asegúrate de ingresar valores numéricos válidos para los cálculos.

Si necesitas más información sobre la aplicación, puedes hacer clic en la opción "Información" del menú lateral.
